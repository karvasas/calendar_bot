# календарь бот (проект для практики)

телеграм-бот на python для интеграции с google calendar. он отправляет напоминания перед встречами и сохраняет историю завершенных событий.

## основной функционал (команды):
* `/start` — запуск авторизации (открывает браузер для входа в ваш google-аккаунт).
* `/status` — проверка статуса авторизации и активности сессии.
* `/set_reminder <минуты>` — настройка времени отправки уведомлений до начала встречи.
* `/week` — просмотр всех встреч на текущую неделю (с понедельника по воскресенье).
* `/next_week` — просмотр расписания на следующую неделю.
* `/history` — вывод списка из 10 последних завершенных встреч.

## инструкция по запуску

### 1. установка зависимостей
сначала вам нужно создать виртуальное окружение и установить необходимые библиотеки:

```bash
# создаем виртуальное окружение
python -m venv venv

# активируем (пример для fish)
source venv/bin/activate.fish

# устанавливаем пакеты
pip install -r requirements.txt
```

### 2. настройка доступа к google api
боту требуется ключ доступа для работы с календарем:
1. перейдите в google cloud console, создайте проект и включите google calendar api.
2. в разделе credentials создайте oauth client id, выбрав тип приложения desktop app.
3. скачайте полученный json-файл, переименуйте его строго в credentials.json и поместите в корневую папку проекта.

### 3. настройка переменных окружения и запуск
создайте в корне проекта файл .env и добавьте в него токен вашего бота, полученный в botfather:
```env
BOT_TOKEN=ваш_токен_от_бота
```

после этого вы можете запустить бота:
```bash
python main.py
```

## структура проекта:
* `database/` — подключение к sqlite и sql-запросы (пользователи, история, уведомления).
* `handlers/` — обработчики команд telegram (/start, /week, /history и другие).
* `services/` — интеграция с google api и фоновый планировщик для проверки календаря.
* `config.py` — загрузка конфигурации и переменных из файла .env.
* `main.py` — главный файл, который инициализирует и запускает бота.
